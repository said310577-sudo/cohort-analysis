# Cohort Analysis Project (Retention Rate)

## Description
This project analyzes user retention (Retention Rate) using cohort analysis techniques. The dataset includes user and event tables extracted from a database. The project focuses on data cleaning, transformation, cohort segmentation, and analysis of user behavior over time with a special focus on Promo vs Organic acquisition channels.

## Data Sources
- Cohort Users table
- Cohort Events table

## Tools & Technologies
- SQL (data extraction, cleaning, joins, cohort building)
- Google Sheets (cohort tables, pivot tables, visualization)
- Excel (.xlsx)

## Process
- Extracted and cleaned data using SQL queries
- Standardized date formats and event logs
- Built cohort groups based on user registration time
- Calculated cohort age and retention metrics
- Aggregated monthly retention data
- Visualized cohort tables in Google Sheets
- Compared Promo vs Organic user cohorts

## Key Focus
- Retention Rate calculation
- Cohort-based user behavior analysis
- Acquisition channel comparison (Promo vs Organic)

## Results
- Identified retention trends across user cohorts
- Found differences in engagement between Promo and Organic users
- Built structured workflow from SQL data extraction to cohort visualization
