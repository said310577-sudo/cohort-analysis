# Cohort Analysis Project

## Description
This project analyzes user retention using cohort analysis based on user registration and activity data. The analysis was performed in Google Sheets and includes cohort segmentation, retention table building, and insights extraction.

## Tools & Technologies
- Google Sheets
- Microsoft Excel (.xlsx)
- Cohort analysis methodology

## Project Structure
- Cohort _analysis_result_Sergienko.xlsx — final dataset used for analysis
- Data sheet — raw user data and events
- Cohort Table — user retention matrix by cohort
- Conclusions — key insights from the analysis

## My Tasks
- Processed and structured raw cohort data
- Built cohort retention table
- Calculated user retention over time
- Analyzed user behavior across time periods
- Extracted business insights from retention patterns

## Results
- Identified user retention drop-off trends
- Analyzed user engagement over time
- Provided insights for improving user retention strategies
